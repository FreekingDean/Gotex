# Gotex
Terminal browsing of reddit written in Go.

##Required Packages
`code.google.com/p/gopass`

 `github.com/jzelinskie/geddit`

##Building
```bash
git clone https://github.com/FreekingDean/Gotex.git
cd Gotex
go build
./Gotex
```
